"# Password-Generator" 

A simple, customizable password generator built with **React**. Generate strong and secure passwords by adjusting length and including numbers or special characters.

"#Features"

- Adjustable password length (5 to 20)
- Option to include:
  - ✅ Numbers
  - ✅ Special characters
- Automatically regenerates password on change
- Fully responsive and styled UI (optional with CSS)

"#Live link"
 [Click here to try it live](https://generatestrongpass.netlify.app/)


"#Built With"
 
React
Vite or Parcel (depending on your bundler)
JavaScript, HTML, CSS
